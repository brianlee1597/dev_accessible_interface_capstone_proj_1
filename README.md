Basic app. (description to come later)

## Run Locally

Quick Note: You need python3 installed on your mac.

If all set, run:

```bash
  $ python3 -m venv venv
```

Power up virtual env:

```bash
  source venv/bin/activate
```

Install Flask:

```bash
  pip install flask
```

export app.py:

```bash
  export FLASK_APP=app.py
```

Finally run flask:

```bash
  flask run
```
